# 🎓 Reflection

## Overview

Shuttle Solutions provided valuable experience applying Human-Centred Design principles throughout a complete UX design process.

---

# What Went Well

- User research-informed design decisions.
- Early sketching encouraged creative exploration.
- Paper prototyping enabled rapid iteration.
- Usability testing identified issues before implementation.

---

# Challenges

The biggest challenge was balancing user needs with interface simplicity.

Design decisions often required prioritising clarity over feature complexity.

---

# Lessons Learned

This project reinforced several important UX principles:

- Design should begin with research rather than assumptions.
- Iteration is essential for improving usability.
- Testing reveals problems that designers often overlook.
- User-centred design produces more effective solutions.

---

# Future Improvements

Given additional time, future iterations could include:

- Digital high-fidelity prototypes
- Accessibility evaluation
- Mobile usability testing
- Larger participant samples
- Integration with real shuttle providers

---

# Final Thoughts

Shuttle Solutions strengthened my understanding of the complete UX lifecycle—from user research to testing—and highlighted the value of iterative, evidence-based design in creating meaningful user experiences.
