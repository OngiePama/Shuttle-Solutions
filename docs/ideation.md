# Design Exploration

Several interface layouts were explored to determine the most intuitive booking flow.

Different approaches focused on:

- Navigation
- Booking flow
- Driver selection
- Payment
- Ratings

---

# Selected Concepts

The strongest concepts prioritised:

- Minimal cognitive load
- Clear navigation
- Transparent booking process
- Simple interactions

---

# Key Takeaway

Exploring multiple ideas before committing to a solution resulted in a more refined and user-centred design.
